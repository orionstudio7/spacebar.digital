Boycott Search

A progressive web application to help users navigate current boycotts. Users can type in a brand they are considering buying, and search common boycotts. Built for my BU METCS 7.01 final project using AngularJS. I hope to edit the page once a month with new boycotts and updates to the brand database using data from the USPTO
