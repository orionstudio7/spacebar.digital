Club Clue

Summary

Have you ever wondered what if Nancy Drew games were gayer and trans-er and indie-er? This is that game! With unique characters, we hope to craft compelling narratives and gameplay based on issues that are important to us.

This point-and-click mystery demo was made for the Trans Representation Jam 2025. It's an attempt to show what we were able to make in around two months while juggling full time work and other responsibilities. This was made using Godot, EgoVenture, and Dialogic. Download Here!

What I Learned and What's Next

This project required me to learn more about how to add external libraries to my games. The EgoVenture framework is a point-and-click engine built into Godot. While I loved a lot of its features, I didn't like it's dialogue framework, Parrot. So, I integrated EgoVenture's point-and-click interaction features with Dialogic's event and dialogue features to make this game come together.

There are two things we would like to add to the demo. The first is accessibility features. Taking my lessons from previous projects, I would like to convert the point and click model into an optional one-button mode, allowing more players to interact with the game without the use of a mouse. I would also like to add text-to-speech options so that low-vision and blind players can play as well.

I also would like to add one more of game locations into the demo. I believe having the player navigate to two different in-game locations and look for clues at each will provide a good summary of what the eventual gameplay will be like.
